# PAD-US-DMS-Scoresheet-Analysis
This repository contains data for analyzing the use of PAD-US GAP Status 1 and 2 lands within the project scoresheet to help rank proposed mitigation projects.
It includes a Jupyter Notebook illustrating some basic stats about how current (April 2025) PAD 1 &amp; 2 lands are distributed across USGS watershed catchments along with statistics of percent PAD 1 & 2 lands compared to water quality, habitat, and hydrology metrics as detailed in catchments that are Target Resource Areas (TRAs).
